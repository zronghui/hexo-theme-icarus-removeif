### 代码

```python
class Solution:
    def divingBoard(self, shorter: int, longer: int, k: int) -> List[int]:
        base, subtract = k*shorter, longer-shorter
        if not k:
            return []
        if not subtract:
            return [base]
        return [base+i*subtract for i in range(k+1)]
```
