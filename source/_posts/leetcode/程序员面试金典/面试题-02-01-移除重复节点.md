### 代码

```python
class Solution:
    def removeDuplicateNodes(self, head: ListNode) -> ListNode:
        s = set()
        dummy = ListNode(0)
        dummy.next = head
        cur = dummy
        while cur.next:
            if cur.next.val in s:
                cur.next = cur.next.next
            else:
                s.add(cur.next.val)
                cur = cur.next
        return dummy.next
```
