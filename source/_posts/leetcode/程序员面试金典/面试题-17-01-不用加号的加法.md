### 代码

```python
class Solution {
    public int add(int a, int b) {
        // 用 Python 此解法会有问题
        while(b != 0) { // 当进位为 0 时跳出
            int c = (a & b) << 1;  // c = 进位
            a ^= b; // a = 非进位和
            b = c; // b = 进位
        }
        return a;
    }
}
```
